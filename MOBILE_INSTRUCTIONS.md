## ‚ùì Troubleshooting

**"The file won't open"**
- Make sure you're opening it with a **web browser** (Safari, Chrome), not a document viewer
- Try saving to Files/Downloads first, then opening from there

**"My changes disappeared"**
- Someone else might have uploaded after you - click "‚¨áÔ∏è Download from Sheets"
- Check if you have internet connection for auto-sync

**"The page looks weird"**
- Try rotating your device (landscape vs portrait)
- Zoom out if things look too large (pinch gesture)
- Make sure you're using a modern browser (Safari, Chrome)

**"Keyboard covers the input fields"**
- Scroll up/down to see fields better
- Rotate to landscape mode for more space
- On iPad, consider using an external keyboard

**"I can't find the file I saved"**
- iOS: Check Files app ‚Üí Downloads or iCloud Drive
- Android: Check Downloads folder or Files app
- Email: The file is still in your email as an attachment
